# Security Policy

## Supported Versions

NoviqWiki v0.1.x receives security fixes during its active development period.

## Reporting a Vulnerability

Please report suspected vulnerabilities privately to the maintainers through the repository security advisory process when available. Do not disclose exploitable details publicly before maintainers have had a reasonable opportunity to investigate and release a fix.

Include:

- Affected version or commit
- Reproduction steps
- Impact
- Any known mitigations

Do not include production secrets, session tokens, passwords, or private user data in reports.
